# Carrot frontend monorepo

A monorepo for Carrot v1 frontend packages.

# Setup 

**Install packages**

On the root folder run

```bash
 pnpm install
```

build packages

```bash
 pnpm build:all
```

## frontend

The carrot UI

To run it

```bash
 pnpm start
```

## UI

The carrot design system with storybook

To run it

```bash
 npm run storybook
```

## SDK

Lib that connects to contracts

## React

Reusable utility functions and hooks
